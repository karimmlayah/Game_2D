# Game_2D
Jeu 2D développé en C avec SDL1.2, incluant animations, décor dynamique, énigmes, mini-jeu et entités interactives. Le joueur explore un monde menacé par la pollution et doit résoudre des défis pour sauver la planète. Le projet intègre sprites, collisions, logique du jeu et boucle optimisée.

A 2D game built using SDL2 and pure C.
Features:
- Player movement
- Sprite animation
- Collision detection
- Enemies & simple AI
- Game loop with delta-time

## Build
make

## Run
./prog
